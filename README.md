# Meal Planning Statistics

A simple web app for weekly meal planning statistics with:

- Meal database with name, category, last cooked date, and cooked counter
- Sortable meal table on all fields
- Default sorting by oldest `last cooked` first
- One-click **Cooked today** action that increments counter and updates date
- Pre-filled Swedish meals

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the app:

   ```bash
   npm start
   ```

3. Open `http://localhost:3000`
